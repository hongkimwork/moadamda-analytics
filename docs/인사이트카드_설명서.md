# 광고 소재 분석 - 인사이트 카드 설명서

> 광고 소재 분석 페이지 상단에 표시되는 5가지 핵심 지표 카드입니다.
> 수많은 광고 소재 중에서 **"지금 당장 봐야 할 소재"**를 자동으로 찾아줍니다.

---

## 1. 🏆 이번 기간 MVP

### 한 줄 설명
**선택한 기간 동안 가장 많은 매출을 만든 광고 소재**

### 쉬운 비유
> 축구팀의 "득점왕"과 같습니다. 이 소재가 우리 매출의 에이스예요.

### 선정 기준
- 해당 기간 내 **총 매출 1위** 소재

### 표시 정보
| 항목 | 설명 |
|------|------|
| 메인 숫자 | 해당 소재가 만든 총 매출액 |
| 보조 정보 | 전환율 (방문자 중 구매한 비율) |

### 활용 방법
- ✅ 이 소재의 **이미지, 문구, 타겟팅**을 분석해서 다른 소재에도 적용
- ✅ 예산 배분 시 우선 고려
- ✅ 비슷한 컨셉의 신규 소재 제작 참고

---

## 2. 💎 객단가 최고

### 한 줄 설명
**구매 1건당 결제 금액이 가장 높은 광고 소재**

### 쉬운 비유
> 백화점 VIP 고객을 데려오는 소재입니다. 한 번 사면 크게 삽니다.

### 선정 기준
- 구매 **3건 이상** 발생한 소재 중
- **객단가(매출 ÷ 구매건수) 1위**

※ 3건 미만은 우연일 수 있어 제외합니다.

### 표시 정보
| 항목 | 설명 |
|------|------|
| 메인 숫자 | 평균 객단가 (1건당 결제 금액) |
| 보조 정보 | 총 구매 건수 |

### 활용 방법
- ✅ **고가 상품** 프로모션에 이 소재 스타일 활용
- ✅ 프리미엄 라인 광고에 참고
- ✅ 이 소재를 클릭한 고객층 분석 (어떤 사람들이 반응하는지)

---

## 3. 🎯 전환율 최고

### 한 줄 설명
**방문자를 구매자로 가장 잘 바꾸는 광고 소재**

### 쉬운 비유
> 영업 사원 중 "클로징 달인"입니다. 매장에 들어온 손님을 놓치지 않아요.

### 선정 기준
- 방문자 **50명 이상**인 소재 중
- **전환율(구매자 ÷ 방문자) 1위**

※ 50명 미만은 통계적으로 신뢰하기 어려워 제외합니다.

### 표시 정보
| 항목 | 설명 |
|------|------|
| 메인 숫자 | 전환율 (%) |
| 괄호 안 | 전체 평균 전환율 (비교용) |
| 보조 정보 | 총 방문자 수 |

### 활용 방법
- ✅ 이 소재의 **랜딩페이지 연결**이 잘 되어 있는지 확인 → 다른 소재에도 적용
- ✅ 소재 이미지/문구가 **실제 상품과 일치**하는지 분석
- ✅ 구매 의향이 높은 타겟에게 노출되고 있을 가능성 → 타겟팅 설정 참고

---

## 4. ⚠️ 점검 필요

### 한 줄 설명
**사람은 많이 오는데 구매가 적은 광고 소재**

### 쉬운 비유
> 손님은 북적이는데 매출이 안 나오는 매장입니다. 뭔가 문제가 있어요.

### 선정 기준
- 방문자 **100명 이상**인데
- 전환율이 **평균의 절반(50%) 미만**인 소재

### 표시 정보
| 항목 | 설명 |
|------|------|
| 메인 숫자 | 전환율 (%) - 낮을수록 문제 |
| 보조 정보 | 방문자 수 (많을수록 손실 큼) |

### 활용 방법
- 🔍 **점검 포인트 체크리스트**:
  1. 광고 이미지와 실제 상품이 다르지 않은가?
  2. 랜딩페이지 로딩이 느리지 않은가?
  3. 가격이 경쟁사 대비 높지 않은가?
  4. 재고 부족/품절 상태가 아닌가?
  5. 타겟팅이 너무 넓지 않은가?

- ⚡ **조치 방안**:
  - 소재 수정 또는 일시 중단 검토
  - 랜딩페이지 개선
  - 타겟팅 재설정

---

## 5. 💡 숨은 보석

### 한 줄 설명
**아직 노출이 적지만 전환율이 매우 높은 광고 소재**

### 쉬운 비유
> 아직 발굴 안 된 신인 가수입니다. 무대에만 서면 팬이 생겨요.

### 선정 기준
- 방문자 **30~200명** 사이 (아직 많이 노출 안 됨)
- 전환율이 **평균의 2배 이상** (잠재력 높음)

### 표시 정보
| 항목 | 설명 |
|------|------|
| 메인 숫자 | 전환율 (%) |
| 보조 정보 | 방문자 수 |

### 활용 방법
- 🚀 **즉시 조치 권장**:
  - 이 소재에 **예산 증액** 검토
  - 노출 범위 확대 테스트
  
- 📈 **기대 효과**:
  - 적은 비용으로 높은 성과 가능
  - 새로운 히트 소재 발굴

## 카드별 요약 비교표

| 카드 | 핵심 질문 | 찾는 소재 | 권장 액션 |
|------|----------|----------|----------|
| 🏆 MVP | 누가 돈을 벌어왔나? | 매출 1위 | 벤치마킹, 예산 유지 |
| 💎 객단가 최고 | 누가 큰 손님을 데려오나? | 고객단가 1위 | 프리미엄 상품에 활용 |
| 🎯 전환율 최고 | 누가 구매를 잘 이끌어내나? | 전환율 1위 | 성공 요인 분석 |
| ⚠️ 점검 필요 | 어디서 손실이 나나? | 방문 多 + 구매 少 | 원인 파악 후 개선/중단 |
| 💡 숨은 보석 | 키워줄 소재가 있나? | 노출 少 + 전환율 高 | 예산 증액 테스트 |

## 참고: 용어 설명

| 용어 | 의미 |
|------|------|
| **전환율** | 방문자 100명 중 몇 명이 구매했는지 (예: 3% = 100명 중 3명 구매) |
| **객단가** | 구매 1건당 평균 결제 금액 |
| **방문자** | 해당 광고를 클릭해서 사이트에 들어온 사람 수 |
| **매출** | 해당 광고를 통해 발생한 총 판매 금액 |
